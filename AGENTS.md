# Multi-Send — Agent Discovery Guide

This project provides agent-native skills for performing batched token transfers on X Layer.

## Available Skills

| Skill | Purpose | When to Use |
|-------|---------|-------------|
| multi-send | Batch transfer of ERC20/Native tokens | User wants to send tokens to multiple recipients in a single transaction or bundle. |

## Quick Start for Agents

Each skill is defined in `skills/` using a `SKILL.md` format that includes:
- YAML Metadata
- CLI instructions
- Usage examples

To use these skills, simply point your tool to the `skills/` directory.
