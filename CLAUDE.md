# Claude Code Instructions — Multi-Send

This project is optimized for Claude Code.

## Usage
Claude Code can discover the multi-send skill via the `.claude-plugin/plugin.json` configuration.

## Key Capabilities
- **Batch Transfers**: Execute multi-recipient transfers on X Layer.
- **Auto-Detection**: Falls back to simple transaction bundling if the MultiSend contract is not deployed.
- **Gas Estimation**: Automatic simulation and gas prediction for batch operations.
